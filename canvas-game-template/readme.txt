Игровой цикл можно использовать для реализации аркадных игр: обработки ввода пользователя, обновления состояния игры, рендеринга и многого другого. Очень удачное решения для анимации чего либо. Скорость движения объектов не зависят от частоты обновления кадров экрана.
По мотивам туториала от ThreePixDroid (в этот раз решил не изобретать велосипед), но с небольшими корректировками.
